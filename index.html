Tentang Sistem

Hardware

  • MCU ESP32 (Dual Core 240 MHz)
  • Gas MQ-135 (8 parameter)
  • Env DHT11 (suhu & kelembaban)
  • Vital MAX30102 (HR & SpO₂)
  • Display OLED 0.96" SSD1306 I²C
  • Buzzer Aktif saat kondisi bahaya

Backend

  • Database Supabase (PostgreSQL)
  • API Supabase REST v1
  • Auth Anon Key (RLS disabled)
  • Kirim Setiap 10 detik dari ESP32
  • Hapus Auto-delete tiap 3 menit (pg_cron)
  • Poll web Setiap 5 detik dari browser

Parameter Gas (MQ-135)

  • CO₂ Karbon dioksida — Bahaya >1000 ppm
  • CO Karbon monoksida — Bahaya >50 ppm
  • NH₃ Amonia — Bahaya >50 ppm
  • Smoke Asap — Bahaya >500 ppm
  • Benzene Senyawa aromatik — Bahaya >5 ppm
  • Alkohol Uap alkohol — Bahaya >800 ppm
  • NOₓ Nitrogen oksida — Bahaya >1 ppm
  • SO₂ Sulfur dioksida — Bahaya >5 ppm

Vital Signs (MAX30102)

  • HR Normal 60–100 BPM
  • HR Lambat <60 BPM (bradikardi)
  • HR Tinggi >100 BPM (takikardi)
  • SpO₂ Normal ≥ 95%
  • SpO₂ Rendah <90% → darurat
  • Deteksi jariIR threshold > 50.000

Legenda Warna

  • Hijau Normal / Aman
  • Kuning Perhatian / Peringatan
  • Oranye Bahaya — Tindakan diperlukan
  • Merah Kritis — Segera evakuasi!

Teknologi

  • Firmware Arduino C++ (ESP32)
  • LibrariesMQUnifiedsensor, DHT, SparkFun MAX3010x, SSD1306
  • Web HTML5 · CSS3 · Vanilla JS
  • Grafik Chart.js 4.4
  • Ikon Phosphor Icons
  • Hosting Vercel (static)
